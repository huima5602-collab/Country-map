# Country Map

这是一个在线世界国家与地区百科静态网站。

## 功能

- 世界地图定位
- 国家搜索与大洲筛选
- GDP 排名
- 国家详情卡片
- 人口、面积、货币、语言、地标展示
- 国家对比功能

## GitHub Pages 在线地址

启用 GitHub Pages 后访问：

```text
https://huima5602-collab.github.io/Country-map/
```

## 部署方法

进入仓库 Settings → Pages → Source 选择 `Deploy from a branch`，Branch 选择 `main / root`，保存后等待几分钟即可上线。
